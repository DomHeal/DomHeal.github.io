How to buld a full stack webapp in Docker

In this blog post, I will go over creating a springboot rest api for backend and a front-end built in React. This blog will cover building simple communications between the frontend and backend and saving data into a Postgres database. It will also go over testing with the use of docker and show quickly you can start your 3-part webapplication with docker-compose.
